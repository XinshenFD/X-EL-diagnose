# Sanitized Code Package

This repository contains a cleaned and anonymized R code package for submission.

## Scope
- Algorithm and model construction scripts.
- Validation and plotting scripts used in the manuscript workflow.
- Shiny platform script (`app.R`) and dependency loader (`library.R`).
- An anonymized sample dataset: `data/sample1_anonymized.xlsx`.

## Notes
- Sensitive text and non-English comments were removed.
- File names were normalized to ASCII where needed.
- Non-essential or unrelated scripts were removed.
